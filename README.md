# Отправка комиксов в Телеграм

Скрипт загружает фотографии комиксов в Telegram канал.

## Как установить
Скачайте репозиторий и установите Python пакеты из `requirements.txt`:
```bash
pip install -r requirements.txt
```
Для взаимодействия необходимо получить их токены. Создайте в папке с проектом файл `.env` и добавляйте переменные с токенами в него, вот так:
```
TELEGRAM_BOT_TOKEN=<ваш телеграм токен>
TELEGRAM_CHAT_ID=<ваш чат айди>
```
Какие переменные `.env` понадобятся:
- TELEGRAM_BOT_TOKEN: необходимо получить в telegram у BotFather
- TELEGRAM_CHAT_ID: необходимо получить в telegram в вашем канале 

## Основной скрипт

**main.py**  
Скачивает фото комиксов (https://xkcd.com/353/). Пример запуска:
```
python main.py
```
